# Project6thSEM Frontend

A Vite + React + TypeScript frontend for the landing page.

## Setup

```bash
cd FRONTED
npm install
npm run dev
```

Then open the local dev server URL printed by Vite in your browser.

## Build

```bash
npm run build
```

## Notes

- The landing page is rendered from `src/pages/LandingPage.tsx`.
- Styles are powered by Tailwind CSS in `src/styles/index.css`.
- The app entry point is `src/main.tsx`.
